使用方法

定期定额复利计算器是计算定期投资时可获得的未来投资收益的工具。

  • 初始本金 : 输入开始金额。
  • 每月投资金额 : 输入每个月要添加的金额。 输入的金额将从第二个月开始添加到本金中。
  • 投资期数 : 输入投资期数，复利将按照输入期数重复计算。
  • 利率 : 输入预期的复利收益率。
  • 复利方式 : 输入计算复利的方式。
  • 计算 : 输出以复利计算的结果。

复利方式

复利方式决定了以前应计利息何时反映在下一次计算中。

  • 年复利 : 年复利中，一年的利息用于计算下一年的利息。
  • 半年复利 : 半年复利中，半年的利息用于计算下一个半年的利息。
  • 每季度复利 : 每季度复利中，每个季度的利息用于计算下一季度的利息。
  • 月复利 : 月复利中，一个月的利息用于计算下一个月的利息。
  • 日复利（360/年） : 日复利中，一天的利息用于计算第二天的利息。这个计算器假设一个月有30天，一年有360天。因此，与实际结果可能存在细微差异。
